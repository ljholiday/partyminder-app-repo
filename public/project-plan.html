Party Minder Project Plan

Project Overview

Project Name: Party Minder
Project Type: Social Event Planning Platform with Embedded Social Networking
Project Owner: Lonn Holiday
Project AI Partner: Leo (ChatGPT)
Purpose: Put the social back into social networks through hosted events.

Scope Statement

In Scope (MVP):

  • User roles: Hosts, Guests, Vendors
  • Unlimited event creation and invitations
  • Guest preference collection (food, allergies, calendar)
  • Custom invite builder (image/logo and event image)
  • RSVP system with email notifications
  • Host dashboard, Admin panel
  • Group chats and post-event conversations
  • Vue frontend with Laravel backend

Out of Scope (for MVP):

  • Food/grocery delivery
  • Premium upgrades or designs
  • Mobile apps
  • Advanced AI features

Work Breakdown Structure (WBS)

  • Planning & Setup: Git, hosting, workflow
  • Frontend (Vue): Invite form, dashboard, profiles, chat
  • Backend (Laravel): Events, invites, users, preferences, messaging
  • Database (MariaDB): Tables for all major features
  • Styling: Shared theme, responsive design, accessibility
  • Email System: Notification templates and logic
  • Testing: Bug fixing, compatibility checks
  • Deployment: Push to production, launch monitoring
